Instructions for use of symbol-less Assembler:
1. Open the Assembler-NoSymbol directory
2. On the command line, enter "Assembler-NoSymbol (directory to .asm file)"
3. The translated .hack file will appear in the same directory as the .asm file

Instructions for use of Assembler (with symbols):
1. Open the Assembler directory
2. On the command line, enter "Assembler (directory to .asm file)"
3. The translated .hack file will appear in the same directory as the .asm file
